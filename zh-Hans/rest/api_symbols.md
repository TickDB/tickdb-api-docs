## 注意事项
- 市场代码不区分大小写
- 交易品种命名规则请参阅数据规范文档
- US、HK、CN 三个股票市场实际上所有已上市的股票产品代码均受支持

## 支持的市场

- **FOREX** - 外汇市场
- **METALS** - 贵金属
- **INDICES** - 市场指数
- **US** - 美国股票市场
- **HK** - 香港证券交易所
- **CN** - A股市场
- **CRYPTO** - 加密货币交易对
