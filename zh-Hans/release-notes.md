## v1.0.1 (2026-03-13)

### 新增功能

- **新增股票市场专属接口**
  - 当日分时数据接口
  - 股票详细信息接口
  - 交易时段查询接口
  - 交易日历查询接口
  - 市场指标数据接口
  - 资金流向数据接口

### 重要改进

- **WebSocket 文档重构**
  - 新增 ping/pong 心跳机制说明
  - 更新 ticker 频道消息格式（按市场类型区分字段）
  - 修正 trade 频道响应格式（包含 trades 数组）
  - depth 频道新增 A 股支持

- **文档内容优化**
  - 重新组织 REST API 导航结构
  - 完善所有接口的请求参数和返回字段说明
  - 更新支持市场标注和示例
  - 统一术语翻译
  - 优化入门指南和数据规范

## v1.0.0 (2026-01-15)

### 首次发布

提供核心 REST API 和 WebSocket 实时订阅，覆盖外汇、贵金属、指数、美股、港股、A 股、加密货币市场。

- **REST API 接口**
  - 交易品种查询
  - 行情快照
  - 历史 K 线
  - 实时 K 线
  - 订单簿
  - 最近成交
  - K 线周期查询

- **WebSocket 频道**
  - ticker 频道
  - depth 频道
  - trade 频道
