## v1.0.1 (2026-03-13)

### 新增功能

- **新增股票市場專屬接口**
  - 當日分時數據接口
  - 股票詳細信息接口
  - 交易時段查詢接口
  - 交易日曆查詢接口
  - 市場指標數據接口
  - 資金流向數據接口

### 重要改進

- **WebSocket 文檔重構**
  - 新增 ping/pong 心跳機制說明
  - 更新 ticker 頻道消息格式（按市場類型區分字段）
  - 修正 trade 頻道響應格式（包含 trades 數組）
  - depth 頻道新增 A 股支持

- **文檔內容優化**
  - 重新組織 REST API 導航結構
  - 完善所有接口的請求參數和返回字段說明
  - 更新支持市場標註和示例
  - 統一術語翻譯
  - 優化入門指南和數據規範

## v1.0.0 (2026-01-15)

### 首次發布

提供核心 REST API 和 WebSocket 即時訂閱，覆蓋外匯、貴金屬、指數、美股、港股、A 股、加密貨幣市場。

- **REST API 接口**
  - 交易品種查詢
  - 行情快照
  - 歷史 K 線
  - 實時 K 線
  - 訂單簿
  - 最近成交
  - K 線週期查詢

- **WebSocket 頻道**
  - ticker 頻道
  - depth 頻道
  - trade 頻道
