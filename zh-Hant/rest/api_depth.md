**GET** `/v1/market/depth`

## 概述
獲取交易品種的實時訂單簿深度（買賣盤）數據。

## 多市場示例

**加密貨幣**：`BTCUSDT`、`ETHUSDT`、`ADAUSDT`
**美股**：`AAPL.US`、`TSLA.US`、`MSFT.US`
**港股**：`700.HK`、`9988.HK`、`3690.HK`

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
     "https://api.tickdb.ai/v1/market/depth?symbol=700.HK&limit=10"
```
