# Errors

## Error Format

```json
{
  "code": 10001,
  "message": "Invalid API Key"
}
```

## Common Errors

| Code  | Description |
|-------|-------------|
| 10001 | Invalid API Key |
| 10002 | Rate limit exceeded |
| 10003 | Invalid parameters |
